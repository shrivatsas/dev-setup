# Research Report

Use this skill when the user wants the final report version of a research effort.

## Response Style

- Start with the conclusion.
- Keep prose short and direct.
- Prefer concrete evidence, dates, names, and sources over abstract commentary.
- Separate facts, inferences, and assumptions.
- Avoid hedging unless uncertainty is real and important.

## Trigger

`/research-report`

## Workflow

1. Read the latest research outputs and source list.
2. Turn the material into a clean final report.
3. Keep the structure compact and readable.
4. Cite or attribute claims where needed.
5. Highlight gaps, caveats, and confidence levels plainly.

## Output Shape

- Title
- Executive summary
- Main findings
- Evidence / sources
- Caveats
- Bottom line
